# 학과 소개의 날 — 교수 면담 배정 시스템 (Professor Interview Scheduling System)

> 대학교 학과 소개의 날 행사에서 학생들이 희망 연구실(교수)을 선택하고, 학과 측에서 실시간으로 세션 단위로 배정 및 관리할 수 있도록 지원하는 웹 기반 플랫폼입니다.

---

## � 주요 🌟 특징 (Features)

1. **학생 모바일 신청 (QR 기반)**
   - 행사장에 띄워진 대형 QR 코드를 스캔하여 바로 접속 (로컬 LAN/와이파이망 지원 지원)
   - 모바일 최적화 UI 적용 (Next.js Viewport 처리 완료)
   - 빠른 연락처, 이름 기반 식별 (개인정보 보호)
   - 1지망~3지망 연구실 실시간 클릭 선택

2. **강력한 관리자 대시보드 (Admin Dashboard)**
   - **실시간 현황 모니터링**: 1~3지망 연구실별 경쟁 현황 및 세션별 잔여 인원 시각화
   - **스마트 자동 배정 알고리즘**: 단 1초 만에 학생들의 1~3지망 우선순위에 따라 3개의 세션에 골고루 분산 자동 배정
   - **결함 탐지 & 경고**: 배정이 되지 않았거나(공강 발생), 지망 외 연구실로 밀려난 케이스를 찾아내어 대시보드 상단 경고창에 표시
   - **조작 안전망 (Fail-safe)**: "결과 공개하기" 토글을 켜기 전까지 학생들은 "조율 중" 대기 화면만 열람 가능

3. **행사 진행 오프라인 지원**
   - **A4 인쇄 최적화 레이아웃 지원**: `/admin/print` 명단
   - **엑셀 CSV 추출 지원**: 학생 명단 및 전체 세션별 배정 데이터 원클릭 다운로드

---

## �️ 기술 스택 (Tech Stack)

- **Frontend**: Next.js 15.2 (App Router), React 19, Vanilla CSS (Custom Design System)
- **Backend**: Next.js Server API Routes (Node.js)
- **Database**: `better-sqlite3` (서버사이드 로컬 `data/interview.db` 저장)
- **Utils**: `qrcode` (결과 QR 자동 송출 모듈)
