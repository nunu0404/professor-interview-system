# 🎓 학과 소개의 날 — 교수 면담 배정 시스템 운영 매뉴얼

안녕하세요! 이 매뉴얼은 **'그대로 따라만 하면'** 행사를 완벽하게 운영하실 수 있도록 작성되었습니다.

---

## 🛠️ 1. 준비물 설치 (최초 1회만 하면 됨)

컴퓨터에 이 프로그램을 실행할 수 있는 기초 환경을 만듭니다.

1.  **Node.js 설치**: [https://nodejs.org/ko/](https://nodejs.org/ko/)에 접속하여 **20 LTS 버전**을 다운로드 후 설치하세요. (그냥 '다음/Next'만 계속 누르시면 됩니다.)
2.  **프로그램 다운로드**: 현재 이 화면 우측 상단의 초록색 **[<> Code]** 버튼 -> **Download ZIP**을 눌러 받은 파일의 압축을 풉니다.
3.  **폴더 열기**: 압축을 푼 폴더 안에 있는 `frontend` 폴더를 찾습니다.
4.  폴더를 오른쪽 마우스를 누른 뒤 '경로로 복사'를 눌러서 폴더 경로를 복사합니다.

---

## 🚀 2. 프로그램 실행 및 종료 방법

행사 당일 아침에 실행하고, 행사가 끝나면 종료하는 방법입니다.

### [실행하기]
1.  명령 프롬프트(윈도우 검색창에 `cmd` 입력) 또는 터미널(맥)을 엽니다.
2.  cd 입력 후 ctrl + v(맥은 cmd + v)로 앞서 복사해둔 경로를 입력 후 엔터를 칩니다.   
3.  다음 명령어를 입력하고 엔터를 칩니다:
    ```bash
    npm install   (※ 처음 한 번만 하면 됩니다)
    npm run dev
    ```
5.  화면에 ` Ready in ...ms`라고 뜨면 성공입니다! 브라우저(크롬 등) 주소창에 `http://localhost:3000`을 입력해 보세요.

### [종료하기] (매우 중요!)
*   터미널(까만 창)에서 키보드의 `Ctrl` 누른 상태로 `C`를 한 번 누르세요.
*   그러면 실행 중이던 서버가 안전하게 멈춥니다. 그 후에 창을 끄시면 됩니다.

---

## 🗓️ 3. 행사 당일 5단계 운영법

### ① 데이터 청소 (시작 전)
*   `http://localhost:3000/admin` 접속 (비번: `openlab2026`)
*   [⚙️ 설정] 메뉴 하단의 [🗑️ 전체 초기화] 클릭! (연습 데이터 삭제)

### ② 학생 신청 받기 (행사 중)
*   메인 화면의 **QR 코드**를 빔프로젝터로 크게 띄워주세요.
*   학생들이 스마트폰으로 찍어서 신청하면 대시보드에 실시간 인원수가 올라갑니다.

### ③ 자동 배정 (신청 마감 후)
*   전체 인원이 다 들어왔다면 [⚡ 자동 배정] 버튼 클릭! (1초 만에 완료)
*   배정 결과 리스트를 눈으로 훑어보고, 화면 위쪽에 빨간색 경고창(공강 발생, 지망 외 인원)이 없는지 확인합니다.
*   배정이 맘에 들지 않거나 누락 케이스가 있다면, 우측 하단의 연구실 선택 드롭다운을 눌러 구멍이 난 학생의 빈자리를 적절히 수동으로 변경(조율)해 줍니다.

### ④ 결과 공개 (최종 확인 후)
*   [⚙️ 설정]에서 [🔓 결과 공개하기] 스위치를 겹니다.
*   학생들에게 다시 QR 코드를 안내하면 본인 배정 결과를 확인할 수 있습니다.

### ⑤ 엑셀 저장 및 인쇄 (종료 후)
*   [🖨️ 배정 명단]을 눌러서 각 연구실별 출석부를 인쇄하세요.
*   [⬇️ 학생 CSV]를 눌러 전체 명단을 엑셀로 보관하세요.

## ❓ 자주 묻는 질문 (Q&A)

*   **Q: QR 코드가 안 읽혀요!**
    *   A: 인터넷 주소(`NEXT_PUBLIC_BASE_URL`) 설정을 다시 확인하고 서버를 껐다 켜보세요.
*   **Q: 학생이 이름을 틀렸대요.**
    *   A: 관리자 대시보드에서 해당 학생을 찾아 직접 이름을 수정하거나 삭제 후 다시 신청하게 하세요.
*   **Q: 비밀번호가 뭐예요?**
    *   A: 관리자는 `openlab2026`, 단순 조회용(교수님용)은 `viewer2026`입니다.
